# Choco Energy CLI

A simple Node.js CLI tool that fetches chocolate data from an API and finds the highest-energy chocolate per gram for a given country.

## Features

- Fetches chocolate data from a public API
- Calculates energy per gram for each chocolate
- Returns the chocolate with the highest energy for the specified country
- Works directly in the console

## Requirements

- Node.js (v14 or higher)
- npm
